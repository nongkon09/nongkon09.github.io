LAB 1

Set static IP address

nmcli connection modify 'Wired connection 1' ipv4.addresses 192.168.59.100/24
nmcli connection modify 'Wired connection 1' ipv4.method manual
nmcli connection modify 'Wired connection 1' ipv4.gateway 192.168.59.2
nmcli connection modify 'Wired connection 1' ipv4.dns 192.168.59.2
nmcli connection modify 'Wired connection 1' ipv6.method disabled
nmcli connection up 'Wired connection 1'
